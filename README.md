# 16S rRNA Gene Sequences for Phylogenetic Analysis

## Overview
This repository contains 16S rRNA gene sequences and phylogenetic analysis files for the study on bacterial isolates from water sources in Ilorin, Nigeria.

## Contents

### Sequences
- **phylogenetic_tree_sequences_supplementary.fasta** - Combined FASTA file with 10 sequences (5 study isolates + 5 reference type strains)
- **STEP3_tree_CORRECTED.nwk** - Neighbor-Joining phylogenetic tree in Newick format
